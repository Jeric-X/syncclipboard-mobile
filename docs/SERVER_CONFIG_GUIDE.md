# 服务器配置功能使用说明

## 📱 功能概述

设置页面现已支持完整的服务器配置和多用户切换功能。

## ✨ 功能特性

### 1. 多服务器管理

- 支持添加多个服务器配置
- 支持编辑现有服务器
- 支持删除不需要的服务器
- 一键切换激活服务器

### 2. 服务器类型支持

- **SyncClipboard 服务器**: 官方独立服务器或客户端内置服务器
- **WebDAV 服务器**: 支持 WebDAV 协议的云存储服务（坚果云、NextCloud 等）

### 3. 连接配置

- 服务器地址配置
- 用户名和密码认证
- 连接测试功能
- 自动同步设置
- 同步间隔配置
- 通知开关

### 4. 智能 UI

- 当前激活服务器高亮显示
- 服务器信息卡片展示
- 空状态提示
- 友好的错误提示

## 🎯 使用步骤

### 添加第一个服务器

1. **打开设置页面**
   - 点击底部导航的 "设置" 标签

2. **添加服务器**
   - 点击 "服务器配置" 区域右上角的 "+ 添加" 按钮

3. **选择服务器类型**
   - **SyncClipboard 服务器**: 如果使用官方独立服务器或桌面端内置服务器
   - **WebDAV 服务器**: 如果使用坚果云、NextCloud 等 WebDAV 服务

4. **填写连接信息**
   - **服务器地址**:
     - SyncClipboard: `http://192.168.1.100:5033`
     - WebDAV: `https://dav.jianguoyun.com/dav`
   - **用户名**: 你的账号
   - **密码**: 你的密码

5. **测试连接**
   - 点击 "测试连接" 按钮验证配置是否正确
   - 成功后会显示 "服务器连接测试成功！"

6. **配置同步选项**
   - **自动同步**: 开启后在后台自动同步
   - **同步间隔**: 设置同步频率（最低 10 秒，推荐 60 秒）
   - **同步通知**: 开启后同步完成时显示通知

7. **保存配置**
   - 点击右上角 "保存" 按钮

### 添加更多服务器

重复上述步骤，可以添加多个服务器（例如：家里的服务器、公司的服务器、云存储等）。

### 切换服务器

1. 在服务器列表中，点击想要激活的服务器卡片
2. 确认切换提示
3. 系统会自动切换到新服务器，并更新同步状态

### 编辑服务器

1. 点击服务器卡片上的 "编辑" 按钮
2. 修改需要更改的配置
3. 可再次测试连接
4. 点击 "保存" 保存修改

### 删除服务器

1. 点击服务器卡片上的 "删除" 按钮
2. 确认删除操作
3. 服务器配置将被移除

## 🔐 安全说明

- 密码使用 SecureStorage 加密存储
- 支持 HTTPS 加密传输
- 建议使用强密码
- 公网服务器务必使用 HTTPS

## ⚠️ 注意事项

1. **同步间隔**
   - 最小值为 10 秒
   - 推荐值为 60 秒
   - 过短的间隔会增加耗电和流量

2. **网络要求**
   - 局域网服务器需在同一网络
   - 公网服务器需能访问互联网
   - WebDAV 通常需要 HTTPS

3. **服务器切换**
   - 切换服务器后会立即停止当前同步
   - 新服务器会从下次同步生效
   - 历史记录保持独立

4. **测试连接**
   - 保存前建议先测试连接
   - 确保服务器地址正确
   - 确保用户名密码正确

## 🐛 常见问题

### Q: 为什么连接测试失败？

A: 检查以下几点：

- 服务器地址是否正确（包括 http:// 或 https://）
- 网络是否能访问该地址
- 用户名和密码是否正确
- 服务器是否在运行

### Q: 如何知道哪个是当前激活的服务器？

A: 激活的服务器卡片会：

- 显示蓝色边框
- 右上角显示 "当前" 标签
- 列表中最明显

### Q: 可以同时使用多个服务器吗？

A: 同一时间只能激活一个服务器，但可以随时切换。

### Q: 删除服务器会删除历史记录吗？

A: 删除服务器配置不会删除本地历史记录，但会断开与该服务器的连接。

### Q: 支持导入/导出配置吗？

A: 暂不支持，未来版本会添加此功能。
