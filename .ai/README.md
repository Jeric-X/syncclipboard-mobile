# .ai 目录说明

这个目录专门为 AI 助手提供上下文信息，帮助在长期开发过程中保持连续性。

## 📁 文件说明

- **`CONTEXT.md`** - AI 的上下文指南，包含工作流程和规范
- **`QUICK_REFERENCE.md`** - 快速参考手册，包含常用代码和命令
- **`README.md`** - 本文件，目录说明

## 🤖 给 AI 的建议

每次新会话开始时：
1. 先读 `CONTEXT.md` 了解工作流程
2. 再读根目录的 `PROJECT_STATUS.md` 了解进度
3. 查看 `SESSION_NOTES.md` 了解上次对话
4. 查看 `TODO.md` 了解待办任务
5. 然后继续工作

## 📝 给开发者的建议

你可以使用以下提示词开始新会话：

```
我继续开发 SyncClipboard Mobile 项目。

请先：
1. 阅读 PROJECT_STATUS.md 了解当前进度
2. 阅读 SESSION_NOTES.md 了解上次对话
3. 阅读 TODO.md 了解待办任务

然后告诉我：
- 上次完成了什么
- 当前应该做什么
- 建议的下一步行动

我的问题/需求是：[在这里描述你的需求]
```

## 🔄 工作流程

```
开始新会话
    ↓
AI 阅读上下文文档
    ↓
AI 总结当前状态
    ↓
讨论并执行任务
    ↓
更新进度文档
    ↓
结束会话
```

## 💡 提示

- 这个目录不会被打包到应用中
- 可以添加更多 AI 辅助文件
- 定期回顾和更新这些文档
